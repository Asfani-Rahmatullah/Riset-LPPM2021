Evaluasi Metode {{metode}}

(Mengukur performa metode sistem rekomendasi berdasarkan metrik Precision dan NDCG, yaitu dengan membandingkan Data Rekomendasi dengan Data Ground-truth)

Target User = {{UTarget}}, Jumlah N = {{topN}}, Nilai Precision = {{nilai_Precision}}, Nilai NDCG = {{nilai_NDCG}}
Persamaan:
`Top(l_(u),N)` `:=` `{l_(u)(1), ... ,l_(u)(N)}`
`Precision(Test_(u),l_(u),N)` `:=` `frac(|Top(l_(u),N) \cap Test_(u)|)(N)`
`NDCG(Test_(u),N)` `:=` `frac(DCG(Test_(u),N))(IDCG(N))`
`DCG(Test_(u),l_(u),N)` `:=` `sum_(n=1)^N 1/log_(2)(1+n) .\Pi (l_(u)(n) \in Test_(u))`
`IDCG(N)` `:=` `sum_(n=1)^N 1/log_(2)(1+n)`
Definisi:
`u` =  target user
`N` =  jumlah rekomendasi
`I` =  daftar hotel
`l_(u)` =  data rekomendasi hotel yang sudah terurut secara descending
     untuk target user u, dimana `l_(u) \in I`
`Top(l_(u),N)` =  Top-N daftar rekomendasi hotel untuk target user `u`
`Test_(u)` =  data ground-truth dari target user `u`
`\Pi (x)` =  fungsi kondisi yang bernilai 1 jika `x` TRUE dan 0 jika FALSE
