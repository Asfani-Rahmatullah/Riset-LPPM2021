Sistem Rekomendasi Hotel berbasis Multi-Kriteria dengan menerapkan Teknik Normalisasi dalam Pendekatan Collaborative Filtering

Aplikasi ini adalah prototipe yang mensimulasikan kerja dan analisa dari sistem rekomendasi. Pengguna aplikasi dapat memilih dari kedua metode yang disediakan di dalam sistem untuk menghasilkan Top-N daftar rekomendasi bagi target user. Daftar rekomendasi diperoleh berdasarkan model metode yang dibangun menggunakan Data Latih dari dataset Rating Multi-kriteria Hotel.

Evaluasi

`DCG(Test_(u),l_(u),N) := sum_(n=1)^N 1/log_(2)(1+n) .&\pi (l_(u) &epsilon Test_(u))` `IDCG(Test_(N)) := sum_(n=1)^N 1/log_(2)(1+n)` `NDCG(Test_(u),N) := frac(DCG(Test_(u),N))(IDCG(N))`
Nilai NDCG = {{nilai_NDCG}}
`Top(l_(u),N) := {l_(u)(1), ... ,l_(u)(n)}` `Precision(Test_(u),l_(u),N) := frac(|Top(l_(u),N) \cap Test_(u)|)(N)`
Nilai Precision = {{nilai_Precision}}

Definisi:

  • u = target user
  • N = jumlah rekomendasi
  • I = daftar hotel
  • lu = data rekomendasi hotel yang sudah terurut secara descending untuk target user u, dimana lu ε I
  • Top(lu, N) = Top-N daftar untuk rekomendasi hotel untuk target user u
  • Testu = data ground-truth dari target user u
`Top(l_(u),N) := {l_(u)(1), ... ,l_(u)(n)}` `Precision(Test_(u),l_(u),N) := frac(|Top(l_(u),N) \cap Test_(u)|)(N)`
Nilai Precision = {{nilai_Precision}}

Sumber data: (https://www.kaggle.com/yelp-dataset/yelp-dataset/version/6)
Detil data: merupakan dataset multi-kriteria yang membahas tentang rating hotel. Dimana dataset yang digunakan dalam penelitian ini sebanyak 25.000 yang terdiri dari 2.595 pengguna, 5.209 item, empat kriteria rating (Overall, Usefull, Funny, Cool).
